# Генератор случайных чисел

Простая веб-страница, которая автоматически генерирует случайные числа от 1 до 100 каждую секунду.

## Возможности

- Генерация случайного числа каждую секунду
- Крупное отображение текущего числа
- Таблица последних 50 сгенерированных чисел с временем генерации
- Счётчик общего количества сгенерированных чисел

## Технологии

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Запуск

Просто откройте файл `index.html` в браузере — никаких зависимостей и сервера не требуется.
