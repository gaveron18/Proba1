# USPP — Активная ВПП (Большое Савино, Пермь)

**Живой адрес:** http://173.249.2.184:8099

Веб-приложение для определения активной взлётно-посадочной полосы аэропорта Большое Савино (ICAO: USPP) на основе данных METAR.

## Возможности

- Автоматическое получение METAR с aviationweather.gov каждую минуту
- Определение активной ВПП (03 или 21) по направлению и скорости ветра
- Отображение компаса с направлением ветра и курсом посадки
- Скорость и порывы ветра в м/с, составляющая встречного ветра
- История текущей сессии (до 30 записей)
- История из базы данных с фильтрацией по периоду и статистикой

## ВПП

| ВПП | Курс |
|-----|------|
| 03  | 030° |
| 21  | 210° |

При штиле или переменном ветре используется предпочтительная ВПП 21.

## Технологии

- **Бэкенд:** Python, aiohttp, SQLite
- **Фронтенд:** HTML5, CSS3, Vanilla JS, SVG-компас
- **Данные:** [aviationweather.gov METAR API](https://aviationweather.gov/api/data/metar)
